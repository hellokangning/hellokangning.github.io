Hbase Store Internals

Nov 16, 2017
HBase
73 word 1 min read

Contents

Overview

  • An HBase table consists of multiple rows.
  • Table will be splited into regions based on rows.
  • A Store corresponds to a column family for a table for a given region.
  • A Store hosts a MemStore and 0 or more StoreFiles (HFiles).
  • The MemStore holds in-memory modifications to the Store.
  • StoreFiles are composed of blocks.

The hierarchy of objects is as follows,

Table -> Region -> Store -> MemStore -> StoreFile -> Block
