被批伪开源！刚刚融资 6 千万美元的 Redis 怎么了？

Redis Labs 宣布 将他们的许可证从使用Commons Clause修改的Apache2更改为Redis源可用许可证（RSAL）。该协议规定，Redis Labs 自研的 Redis 模块 - RediSearch，Redis Graph，ReJSON，ReBloom 和 Redis-ML 将不再是开源软件，而是源码可用（source available）级别。

Redis-dump-go - A fast Redis dump tool
