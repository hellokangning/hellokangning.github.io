这两天上了一堂水平思考课，所实话，比上班还累！但跟着老师的思路，加上及时的课堂练习，最后是收获满满。

课程之后总感觉要再巩固一下，遂成此文。

简而言之，这是一堂关于创造性思考的课程。有关课程的起源、作者的简介，这里一概略过，直接上干货。

创造性思考过程

  1. 确定一个问题、挑战和机会。仔细地定义这个思考焦点。
  2. 选择一个或多个水平思考工具，并使用他们产生新的想法和概念。
  3. 从所有的主意和概念进行收获，从而获得有组织的、丰富的收获。进一步处理主意，使之变得有用和可行。
  4. 选择新的主意和概念进一步发展。

焦点

焦点定义的步骤：

  1. 列出一份创造性思考的焦点列表。
  2. 选择一个焦点。
  3. 对这个焦点进行重新定义。
  4. 将这个焦点进行精炼。

焦点的两种类型

焦点可以分为：

  1. 区域焦点，关注“在哪里”。
  2. 目标焦点，关注“为什么”。

区域焦点指出我们想在哪一领域获得创意，而不是为什么获得创意。区域焦点促使我们对任意一个领域进行思考，而不是局限于解决问题。比如：

  • 关于会议室内种的想法。
  • 关于顾客的新主意。

目标焦点清楚地表示思考的目的，比如要解决某个问题，或者达成某项任务。目标焦点应该包含一个动词。比如：

  • 我们如何提高团队的士气？
  • 我们是可以使用哪些办法来减少用纸？

如果要寻找某个领域的新创意，使用区域焦点；如果想获得某种结果或解决某个问题，使用目标焦点。

重新定义焦点

焦点不存在唯一性，与其匆忙产生创意，不如先从好几个不同的角度仔细考虑问题。

这里用到的方法是：

  • 向上问 WHY，寻找更宽阔的视野。
  • 向下问 WHAT，收缩一个焦点。

产生主意

概念提取

概念是模糊的、不具体的，而主意则是实施概念的方法，更为具体。

概念提取有两个工具，分别是概念三角和概念扇。

概念扇的步骤：

  1. 定义出目标。
  2. 从目标往回倒推出宽泛的概念，或者我们能达到目标的一般方向。
  3. 从每个方向倒推，找出能推动我们朝着这一方向前进的概念。
  4. 把步骤3找到的概念罗列出来，然后分别寻找具体的方法来实施这个概念。

挑战

挑战的步骤:

  1. 选择一个焦点。
  2. 列出与焦点相关的、我们认为理所当然的事情。
    • 那里有什么？列出可观察到的性质和特征。
    • 现行的思考是什么？找出围绕焦点的先行思考模式。
  3. 从我们认为理所当然的事情中选一项。
  4. 运用一个或多个挑战工具：为什么/CBA。
  5. 从步骤4像处的主意中选择一个予以实施，或者运用另一个思考工具来获得更多的主意。

那里有什么？

  • 目标对象的整体或局部。
  • 情境的整体或局部。
  • 系统或流程的整体或局部。

现行思考是什么？

  • 主导观点
  • 边界
  • 假设
  • 基本要素
  • 避免因素

随机输入

随机输入的步骤：

  1. 选择一个焦点。
  2. 随机获得一个刺激物。
  3. 将随机选择的刺激物与焦点摆放到一起。
  4. 想出各种注意把刺激物和焦点联系起来。
  5. 从步骤4想出的主意中选择一个予以实施，或使用其他工具获得更多的主意。

激发和运动

激发和运动的步骤：

  1. 选择一个焦点。
  2. 列出你认为关于这个焦点的理所当然的事情。
  3. 使用激发技巧创建激发。
  4. 选择最大胆的那个激发。
  5. 使用运动技巧，产生出符合焦点的新创意。
  6. 收获所产生的创意，进行处理和评估。

有好几种建立激发的方法，包括：

  1. 摆脱：取消或者去掉你认为的理所当然。
  2. 反向：逆反常规方向的思考。
  3. 许愿：创造一个白日梦（如果。。。不好吗？）
  4. 夸张：提出一个超出常规的尺度。

如果我们对激发进行运动，我们就可以把它视为 PO，并以它为跳板，获得新的创意。

以下是3个运动工具：

  1. 时刻观察：形象化地想象 PO 之后一刻接一刻的情况，从中运动出新的创意。
  2. 提炼概念：从 PO 的表达中提炼一个概念，获得新的创意。
  3. 积极方面：列出 PO 的种种好处，从而获得新的创意。

处理和评估

处理

处理旨在保持原创性的同时，又增强实用性。有两个常用的处理技巧：

  1. 修正：运用现实生活中的约束条件，对主意进行修整。
  2. 加强：找出主意的有点，想办法加强。

回到我们的焦点：提升团队士气。我们想到的主意是设立专门的空间来开展团队活动。但约束条件是很难让所有成员都同时离开岗位参加活动，修整之后的办法可以是:

  • 电话会议
  • 在晚上开展活动
  • 每次的活动时间持续半天。

加强主意的方法就是考虑关键的相关人群。例如我们以加强团队精神为焦点，主意是颁奖宴会。

相关人群 观点 加强主意的办法
领导 希望鼓励成员达到团队目标 给每个成员一个特别的奖励
成员 希望自己的贡献得到承认 让CEO褒奖
成员的家属 希望自己对团队的支持得到认同 邀请家属参加晚宴

评估

评估就简单多了，万能的四象限法。分别从实行难易程度和效果好坏来评估主意。
